name: Pas de co-auteur dans les commits
description: Ne jamais ajouter Claude comme Co-Authored-By dans les commits git
type: feedback
---

Ne jamais mettre "Co-Authored-By: Claude..." dans les messages de commit git.

**Why:** L'utilisateur ne veut pas que Claude apparaisse dans l'historique git.
**How to apply:** Tous les commits doivent être propres, sans mention de Claude.
