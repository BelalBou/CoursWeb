# CoursWeb Studio

Application Windows pour lire le parcours `CoursWeb` dans une interface moderne.

## Contenu inclus

L'application embarque les dossiers de cours dans cet ordre :

1. `typescript`
2. `nextjs`
3. `nestjs`
4. `prisma`
5. `postgresql`
6. `linux`

Le dossier `mon-premier-projet` n'est pas inclus volontairement.

## Développement

```powershell
npm.cmd start
```

## Build

```powershell
npm.cmd run build
```

## Installateur Windows

```powershell
npm.cmd run dist
```

Le résultat sort dans `release`.
